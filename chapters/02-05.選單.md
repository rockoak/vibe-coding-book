## 二、Cursor 上方選單各功能說明

你列出的這一排，基本上是 **Cursor = VS Code + AI 的選單系統**  
下面用「實務導向」幫你快速理解 👇

***

### 📁 檔案 (F / File)

**專案與檔案管理**

常用功能：

*   新增 / 開啟 / 儲存檔案
*   開啟資料夾（專案）
*   關閉編輯器、最近使用的專案
*   匯出、重新載入視窗

👉 跟 AI 無直接關係，但是所有工作的入口

***

### ✏️ 編輯 (E / Edit)

**文字與程式碼操作**

常用功能：

*   復原 / 重做
*   剪下 / 複製 / 貼上
*   搜尋 / 取代
*   多游標、區塊編輯

👉 AI 生成後「人工微調」最常用這一欄

***

### 🔍 選取項目 (S / Selection)

**選取文字的進階技巧**

例如：

*   擴展選取範圍（語法層級）
*   選取相同單字
*   選取整個函式 / 區塊

👉 搭配 AI 很好用  
例：選一個 function → Ask AI 改寫

***

### 👁️ 檢視 (V / View)

**畫面與工具顯示**

常用：

*   側邊欄（Explorer / Search / Git）
*   Command Palette
*   面板顯示（終端機 / Output）
*   縮放、編輯器配置

👉 常用來開啟 **AI Chat / AI Panel**

***

### 🧭 移至 (G / Go)

**快速導航用**

例如：

*   移至定義
*   移至實作
*   移至符號（function、class）
*   快速跳檔案

👉 看 AI 產生的程式碼時，用來「追蹤結構」

***

### ▶️ 執行 (R / Run)

**程式執行與除錯**

功能：

*   執行程式
*   除錯（breakpoint）
*   啟動設定

👉 AI 生碼 → 這裡測試對不對

***

### 🖥️ 終端機 (T / Terminal)

**命令列操作**

例如：

*   開新 Terminal
*   執行 CLI、Python、npm、git
*   看 log、跑腳本

👉 Cursor 很常：  
**AI 生指令 → 你在 Terminal 跑**

***

### ❓ 說明 (H / Help)

**說明與版本資訊**

包含：

*   Cursor/VS Code 說明文件
*   快捷鍵列表
*   回報問題
*   關於 Cursor

👉 查快捷鍵時很好用
