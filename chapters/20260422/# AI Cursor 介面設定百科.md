## 1. 活動欄如何從橫向變回縱向

> **提問：AI Cursor 如何讓活動欄橫的變成值的？**

### 💡 回答：
要在 AI Cursor 中將「活動欄」（Activity Bar）的位置改回垂直排列，有以下兩種最直接的方法：

#### 方法 A：滑鼠右鍵（最快）
1. 將滑鼠移動到目前「橫向活動欄」的**空白處**。
2. 點擊 **滑鼠右鍵**。
3. 選擇 **Activity Bar Position** -> **Side**。

#### 方法 B：進入系統設定
1. 按下快捷鍵 `Ctrl + ,` (Windows) 或 `Cmd + ,` (macOS)。
2. 搜尋 `workbench.activityBar.location`。
3. 在下拉選單中選擇 **default**。

## 2. Icon Click Behavior 與 Location 功能詳解

> **提問：1. workbench, Activity Bar : icon Click Behavior 的功能，有 toggle, focus 選項。 2. workbench, Activity Bar : Location，有 Default、Top、Bottom、Hidden 選項。**

### 💡 回答：

#### 一、Icon Click Behavior (圖示點擊行為)
這決定了當你點擊一個「已經處於開啟狀態」的圖示時，介面要怎麼反應：
*   **`toggle` (切換)**：點擊後側邊欄會「縮回去」（關閉）。適合想快速隱藏面板的人。
*   **`focus` (聚焦)**：點擊後側邊欄會「留著」，只是把輸入游標移進去。適合不希望面板意外消失的人。

#### 二、Location (顯示位置)
這決定了那一排功能圖示（檔案、搜尋、AI）要放在哪裡：
*   **Default**：傳統垂直側邊欄（左側或右側）。
*   **Top**：水平放在側邊欄上方（Cursor 預設常用，較省寬度）。
*   **Bottom**：水平放在側邊欄下方。
*   **Hidden**：完全隱藏（需記住快捷鍵才能切換功能）。

## 3. 圖解：何謂「點擊已開啟的圖示」

> **提問：其中的文字 '點擊已開啟的圖示'，這一部分我理解不瞭，請用圖示的方式說明一下。**

### 💡 回答：
「點擊已開啟的圖示」是指：當某個功能（例如：檔案總管）**已經顯示在螢幕上**，且該圖示已經是**亮起來**的狀態，你又去按了它一下。

### 視覺行為圖解


| 設定值 | 視覺流程 (操作步驟) | 最終結果 | 比喻 |
| :--- | :--- | :--- | :--- |
| **toggle** | `[面板開著]` ➜ **再點一次圖示** | **面板消失 (關閉)** | 像**原子筆**，按一下出芯，再按一下縮回。 |
| **focus** | `[面板開著]` ➜ **再點一次圖示** | **面板不動 (保持)** | 像**電梯按鈕**，已在該樓層，再按也只是確認。 |

## 4. 什麼是 Secondary Side Bar？

> **提問：什麼是 Secondary Side Bar ? 請用圖示說明**

### 💡 回答：
**Secondary Side Bar (次要側邊欄)** 是編輯器右側的另一個獨立存放區。它讓你可以同時在左、右兩邊放置不同的面板。

#### 視覺佈局圖解：
```text
+--------+-----------------------+--------+

|  Main  |                       |  Sec.  |
|  Side  |       Editor          |  Side  |
|  Bar   |      (程式碼區)        |  Bar   |
|        |                       |        |
| 📁 🔍  |                       | 🤖 🏗️  |
+--------+-----------------------+--------+
 (左側主要)                     (右側次要)
```

#### 為什麼要用它？
在 Cursor 中最經典的用法是：
*   **左側 (Main)**：放「檔案總管 📁」，用來找檔案。
*   **右側 (Sec.)**：放「AI Chat 🤖」，用來寫代碼。
*   **好處**：你不需要在檔案清單和 AI 對話框之間切換，兩者可以並存看個夠。

#### 如何開啟/關閉？
*   **快捷鍵**：`Ctrl + Alt + B` (Windows) / `Cmd + Opt + B` (Mac)。
*   **操作**：你可以直接把任何圖示（例如：搜尋、AI）用滑鼠從左邊拖曳到最右邊，它就會自動開啟次要側邊欄。
