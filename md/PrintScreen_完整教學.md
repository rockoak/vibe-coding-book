# Print Screen（螢幕截圖）完整教學

本文**僅介紹 Print Screen（截圖）功能**，**與 Cursor 或 vibe-coding-book 無關**，可獨立使用。

---

## 一、什麼是 Print Screen？

Print Screen（常標示為 **PrtSc / PrtScn / PrintScrn**）是鍵盤上的截圖鍵，用來把螢幕畫面擷取成圖片，供後續貼上或儲存。

---

## 二、Windows 常見 Print Screen 用法

### 1️⃣ `Print Screen`
- 功能：擷取 **整個螢幕**
- 結果：存放在「剪貼簿」
- 用法：
  1. 按下 `Print Screen`
  2. 開啟 Word / Paint / Line / Cursor / Slack
  3. `Ctrl + V` 貼上

---

### 2️⃣ `Alt + Print Screen`
- 功能：只擷取 **目前作用中的視窗**
- 結果：存放在剪貼簿
- 適合：只想抓單一程式畫面（例如瀏覽器、IDE）

---

### 3️⃣ `Windows + Print Screen`
- 功能：擷取 **整個螢幕並自動存檔**
- 儲存位置：
  ```
  圖片 > 螢幕擷取畫面 (Screenshots)
  ```
- 優點：不需要手動貼上

---

### 4️⃣ `Windows + Shift + S`（強烈推薦 ✅）
- 功能：啟動 **螢幕擷取工具**
- 可選模式：
  - 矩形擷取
  - 自由形狀擷取
  - 視窗擷取
  - 全螢幕擷取
- 結果：截圖後存入剪貼簿，可立刻貼上

👉 **工程師 / 文件撰寫者最常用方式**

---

## 三、截圖後圖片在哪？

| 截圖方式 | 圖片位置 |
|--------|----------|
| Print Screen / Alt + PrtSc | 剪貼簿（需自行貼上） |
| Win + PrtSc | 圖片/螢幕擷取畫面 |
| Win + Shift + S | 剪貼簿 |

---

## 四、常見問題

### Q1：按了 Print Screen 但什麼都沒發生？
✅ 正常，圖片在「剪貼簿」，請用 `Ctrl + V` 貼上

### Q2：筆電沒有 Print Screen？
✅ 通常是：
```
Fn + Print Screen
```

### Q3：截圖可以直接貼到 Markdown 嗎？
✅ 可以：
- 截圖 → 貼到編輯器（如 Cursor / VS Code / Obsidian）
- 或先存成圖片再引用

## 五、實務建議（懶人版）

- ✅ **精準截圖**：`Win + Shift + S`
- ✅ **大量截圖存檔**：`Win + PrtSc`
- ✅ **文件 / 教學寫作**：搭配螢幕擷取工具最順
